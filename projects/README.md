# Projects Directory

这个目录用于存放视觉设计作品（子网站）。

## 目录结构

每个项目应该放在独立的文件夹中：

```
projects/
├── project-name-1/
│   ├── index.html
│   ├── thumbnail.png  (用于展示的缩略图)
│   └── ... (其他项目文件)
├── project-name-2/
│   ├── index.html
│   ├── thumbnail.png
│   └── ...
```

## 添加新项目

1. 在 projects/ 下创建新文件夹
2. 将你的网站文件放入该文件夹
3. 准备一张缩略图 (建议尺寸: 800x600px)
4. 在 visual-design.html 中添加项目卡片
