# SEO 优化技能

## 目标读者
年龄 25-40 岁，互联网从业者、体制内转型者、自由职业者

## 标题优化
- 长度：40-60字符（中文20-30字）
- 包含核心关键词
- 清晰传达价值

示例：
- ❌ "我的一些想法"
- ✓ "离开体制内的五年反思：从迷茫到找到方向"

## 摘要生成
- 长度：150-160字符（中文80-100字）
- 包含1-2个关键词
- 准确概括内容

模板：
"本文分享[主题]，通过[方法/案例]，帮助你[价值]。适合[读者]。"

## URL Slug 优化
- 使用英文单词，连字符分隔
- 包含核心关键词
- 3-6个单词
- 避免停用词

## Kevin 常用关键词库
- 职业发展：体制内离职、职业转型、GCDF、职业规划师
- AI应用：Claude、ChatGPT、AI工具、提示词工程
- 个人成长：骑行减重、知识管理、时间管理
- 投资：科技股、价值投资、腾讯、小米

## 发布前 SEO 检查清单
- [ ] 标题长度合适且包含关键词
- [ ] URL slug 简短且语义清晰
- [ ] 摘要准确概括内容且吸引人
- [ ] 分类和标签准确
- [ ] 文章长度 >800 字
- [ ] 至少 2 个内部链接
- [ ] H2/H3 标题结构清晰
- [ ] 首段包含核心关键词
